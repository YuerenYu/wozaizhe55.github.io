234. Palindrome Linked List

Posted on 2019-07-22 Edited on 2019-08-10 In leetcode
Symbols count in article: 933 Reading time ≈ 1 mins.

判断单链表是不是回文，这个问题肯定是要遍历链表的，时间复杂度是O(n)，主要看空间复杂度。

思路一：空间复杂度O(n)

  1. 将原链表逆序，保存为一个新的链表，然后遍历新旧链表，逐个元素比较。

  2. 遍历链表，将节点Val保存到数组中，然后遍历数组，逐个元素比较。

以上两种思路时间复杂度、空间复杂度都是O(n)，空间复杂度还有优化的空间。

思路二：空间复杂度O(1)

可以将链表一分为二，其中一部分逆序，然后将这两部分逐个节点比较，空间复杂度为O(1)，比较结束后将原链表复原。golang实现如下：
