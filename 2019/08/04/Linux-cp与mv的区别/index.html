Linux cp与mv的区别

Posted on 2019-08-04 In Linux
Symbols count in article: 271 Reading time ≈ 1 mins.

现在我们当前目录下有个可执行文件login，我们看一下它的信息：

mv

现在我们用mv命令将这个文件移动到/usr/bin中

因为当前不是root用户所以前边加了sudo，看一下执行的结果：

可以看到用户、组、创建时间与mv前保持一致。

cp

再来看一下cp，用cp命令将这个文件复制到/usr/bin中

很明显，用户、组、创建时间发生了变化。

总结

mv只是移动了文件，文件内容文件属性均与之前保持一致；cp是复制了文件内容，重新创建了文件属性。
