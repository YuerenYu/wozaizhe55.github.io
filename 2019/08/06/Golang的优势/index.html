Golang的优势

Posted on 2019-08-06 Edited on 2019-08-10 In Golang
Symbols count in article: 826 Reading time ≈ 1 mins.

问题背景

最近在面试，经常会被问到为什么选择Golang，Golang的优势体现在哪里。作为一个Goper这个问题回答的不是那么流畅，特在此总结一下。

Golang

Go is an open source programming language that makes it easy to build simple, reliable, and efficient software. — golang.org

速度快

Golang在执行效率和开发效率之间做到了协调，性能强劲，开发效率又不弱于python等动态语言。

执行速度快

Golang的执行效率虽然不及C、C++，但强于Java、C#，远远强于python这类动态语言。

开发速度快

Golang 拥有更智能的编译器，只关注被直接引用的库，不像Java、C和C++那样，要遍历依赖链中所有依赖的库，又不像动态语言那样缺少编译过程，类型错误无法捕捉。Golang编译极快，几乎感受不到编译过程。

并发友好

现代计算机都拥有多个核心，但是大部分编程语言都没有有效的工具让程序可以轻易利用这些资源，需要编写大量同步代码来利用多核，很容易出错。Golang一个很重要的特性便是在系统层面支持并发，简化了并发程序的编写。

goroutine

goroutine很像线程，但是它占用的内存远少于线程，使用它需要的代码也很少，一台普通的PC就能支持百万级别的goroutine调用。

channel

channel是一种数据结构，可以让goroutine之间进行安全的数据通信。channel可以帮助用户避免其他语言里常见的共享内存访问的问题。

简单

Golang仅有25个关键字，但是表达能力极强，几乎涵盖大多数编程特性：继承、多态、重载等。Golang的类型也非常简单，不仅支持常见的int、string，还可以像C一样自定义结构体。
