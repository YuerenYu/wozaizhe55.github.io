哪吒之魔童降世 观后感

Posted on 2019-08-20 Edited on 2019-08-21 In 观后感
Symbols count in article: 1.6k Reading time ≈ 1 mins.

《哪吒之魔童降世》火爆了今年的暑期档，随着口碑的不断发酵，票房一路飙升，突破了40亿大关紧追《复仇者联盟4：终局之战》。豆瓣评分8.6，淘票票评分更是高达9.5，而我终于按耐不住，为其贡献了票房。

第一感受

这部电影看的值！无论从剧情还是特效，都值得去电影院观看。剧情的讲解分析，这里不作介绍，感兴趣的同学可以自行搜索。特效解读我也不是专业的，可以看一下大佬们的解读文章，这里重点提一下我的观影思考。

思考

心理学家阿德勒在《自卑与超越》一书中提到：叛逆的孩子之所以叛逆是为了要引起别人的注意，叛逆不是他的目的，只是表达自己缺乏爱的方式而已，人是需要被爱的。这个观点在《哪吒之魔童降世》中体现的淋漓尽致。分别从以下分别从几个方面分析：

  • 主人公：哪吒
  • 父母：李靖夫妇
  • 社会：太乙真人、敖丙、陈塘关百姓

主人公 哪吒

一出生的哪吒便在庭院里暴走，吓坏众人，这也是陈塘关百姓对其存在偏见的根本原因，加之陈塘关一直饱受妖怪困扰，也就有了后来百姓对哪吒的排斥。从哪吒的角度来看，他自己并不知道出生就暴走的事实，他知道的仅仅是百姓对他始终存在恶意，却不知道是为什么。

人之初，性本善。哪吒在心性还能够自控的时候其实是很善良的，对他没有敌意的小朋友，他对别人也很好，而戴着有色眼睛看待他的村民，一直饱受他的捉弄。这也证明了观点：你如何对待别人，别人也将如何对待你。

哪吒在得知自己是“灵珠”转世的时候，主动担当起拯救百姓的重任，追捕海夜叉，救回孩童。村民深受第一印象毒害，认为哪吒还是那个净惹事生非的魔王，非但没有对他英雄救人的行为认可，还恶言相对，最终哪吒向其大打出手。

哪吒内心是渴望得到认同与关爱的，在江山社稷图中，殷夫人给哪吒讲那一番话的时候，他攥紧了拳头，这才有了上边追捕海夜叉想为村民做事情的想法。

李靖夫妇担任镇守陈塘关，公务繁忙，无暇照顾年幼的哪吒，缺少父母关爱的哪吒也对世人充满敌意，认为别人都不关心他。然而影片中，小猪座骑的镜像让他看到了父母是始终爱他的，知道自己是被爱的，内心中所有的恶意、不满便消失而去，唤起了责任感、使命感，将影片推向高潮。

成熟的哪吒对自己的能力有清晰的认识，乾坤圈能够压制心魔，同时也会束缚能力，为了对抗敖丙，哪吒知道自己无法驾驭完全状态，没有全开，将乾坤圈缩小成一个手镯，戴在手上，释放了能力的同时也束缚着心魔。

至此，哪吒的分析就结束了。总的来说，哪吒是一个内心渴望被认同、被爱的孩子，他也想完成大的使命，但是不愿被误会，一旦误会解开，知道父母一直深爱着自己，哪吒变得非常成熟、懂事，这便是爱的力量吧。

父母

不得不说剧中的哪吒是幸运的，有李靖夫妇这样的父母，我觉得现实中的父母可以借鉴一下剧中哪吒父母的教育方式。

得知哪吒是魔丸转世的时候，依旧拼死保护哪吒，为了在天雷中救下哪吒，不惜用自己的生命做交换，父母真的好伟大！

哪吒做错事的时候，严厉惩罚，但又让哪吒拜太乙真人为师学习本领。罚归罚，正确的引导还是要做的。

哪吒不好好学习本领时候，殷夫人殷切开导，搬事实，讲道理，而不是一味的责怪。

哪吒被误会的时候，没有急于责罚哪吒，而是先调查清楚事情真相，得知哪吒是被误会的时候，挨家挨户向村民解释，并恳请村民给哪吒庆生。

有这样懂得良好教育方式的深爱着自己的父母，不得不说哪吒是幸运的。

社会

师父太乙真人在剧中扮演了一个搞笑的角色，在欢声笑语中将本领教给了哪吒。

朋友敖丙对哪吒的认同，给哪吒以极大的信心，以至于得知自己将要办庆生宴的时候，立马写了请柬给敖丙送过去，顺手还画了路线图。

村民一直戴有色眼睛看哪吒，哪吒也不给他们好脸色，时常搞一些恶作剧整村民。可见第一印象是多么的重要。

后记

希望当一个叛逆孩子出现的时候，都能得到正确的引导吧！

叛逆不是目的，爱才是。

背景

算法中最长公共子序列(LCS)， 最长公共子串(LCSub)，最长递增子序列(LIS)是动态规划的重要应用，比较难，却异常受到各大公司笔试的青睐，故在此总结一下。

最长公共子序列

最长公共子序列相当于求两个序列的交集，不需要公共序列不要求连续。这里咱们求最长公共子序列的长度，如果要输出其中一个值的话需要额外的保存状态信息，这个以后再说。比如：

则A、B的最长公共子序列为ABD，输出长度。

分析

设：

$X_i=(X_1,X_2,…,X_i)$表示序列$X$的前$i$个字符$(1 \leq i \leq m)$（前缀）

$Y_j=(Y_1,Y_2,…,Y_j)$ 表示序列$Y$的前$j$个字符$(1 \leq j \leq n)$(前缀)

$Z_k=(Z_1,Z_2,…,Z_k)$ 表示$(X,Y)$的最长公共子序列$(LCS(X,Y))$

  • 若(最后一个字符相同)，容易得出：,显然有, 此时问题转化为求, 并且有：

  • 若$X_m \neq Y_n$ 由于$Z \in LCS(X_m, Y_n)$, 必有：$Z_k = X_m$ or $ Z_k = Y_n$成立

    • 当$Z_k = X_m$ 时 $LCS(X_m, Y_n) = LCS(X_{m-1}, Y_n)$

    • 当$Z_k = Y_n$时$LCS(X_m,Y_n) = LCS(X_m, Y_{n-1})$

  • 因此 $LCS(X_m, Y_n)$转化为三个子问题：

    • $LCS(X_{m-1}, Y_{n-1}) $

    • $LCS(X_{m-1}, Y_n)$

    • $LCS(X_m, Y_{n-1})$

题外话

动态规划(Dynamic programming),看到programming相信好多同学和我一样，瞬间想到的是编程，直到我在算法导论上看到动态规划这章，开篇没几行就提到programming不是编程的意思，这里是指表格！所以动态规划的问题离不开辅助表格，常见的有一维数组，二维数组作为辅助表格，多数情况下二维数组可以优化到一维数组，这个要看具体问题。一些被奉为经典的书籍还是要读一读的，有时候简单的一句话能使自己少走不少弯路。

最长公共子串

给定两个字符串:

求得$A,B$的最长公共子串长度，这里最长公共子串为$”ing”$, 其长度为$3$.

分析

最长公共子串和最长公共子序列是非常相似的，只是最长公共子串要求子串是连续的。设：

$X_i=(X_1,X_2,…,X_i)$表示子串$X$的前$i$个字符$(1 \leq i \leq m)$

$Y_j=(Y_1,Y_2,…,Y_j)$ 表示子串$Y$的前$j$个字符$(1 \leq j \leq n)$

$Z_k=(Z_1,Z_2,…,Z_k)$ 表示$(X,Y)$的最长公共子串$(LCSub(X,Y))$

  • 若$X_m = Y_n $ 则 $Z_k = Z_{k-1} + 1$
  • 若$X_m \neq Y_n$ 则 $Z_k = 0$

$len(Z_k)$ 即为所求。

初始状态

最长公共子串同样借助二维辅助数组$dp[m+1][n+1]$和最长公共子序列的初始状态是一样的.

最长递增子序列

最长递增子序列(LIS)是给定一个数组$nums$, 找出最长的单调递增子序列(不一定连续，但是相对顺序不可打乱)的长度，例如:

$nums$ 的最长递增子序列为${2,3,5}$, 其长度$3$ 即为所求。

解法1：最长公共子序列

可以对$nums$ 进行排序 得到$nums1$ 然后可以转化为求$nums, nums1$ 的最长公共子序列问题。

解法2：动态规划

设：

$X_i = (X_1,X_2,…,X_m)$ 表示以$i$结尾的递增序列元素$(1 \leq i \leq m)$, 则有 $LIS(i) = max{LIS(j)} + 1, (j < i $ and $ nums[j] < nums[i])$

所以求$LIS(i)$ 的时候 我们需要遍历$nums $前$ i-1$ 个元素 ，时间复杂度为$O(n^2)$, $n$ 为$nums$的长度。
